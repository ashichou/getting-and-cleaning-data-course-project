# Code Book

This code book describes all of the data fields in the `tidy.txt` file.
See the `README.md` file of this repository for background information on this data set.

## Source

* `features.txt`: List of features.
* `activity_labels.txt`: Names and IDs for each of the 6 activities.

* `X_train.txt`: Train set.
* `subject_train.txt`: 
* `y_train.txt`: Training labels.

* `X_test.txt`: Test set.
* `subject_test.txt`: 
* `y_test.txt`: Test labels.

More information about the features is available in `features_info.txt`

## Identifiers
- `activity`

	Activity identifier, string with 6 possible values: 
	- `WALKING`
	- `WALKING_UPSTAIRS`
	- `WALKING_DOWNSTAIRS`
	- `SITTING`
	- `STANDING`
	- `LAYING`

## Transformations

The following is the steps to clean the data 

1. Load all he training and test sets and merge in a single data set "mergeData".
2. Name all columns.
3. Remove all columns that does not contains "mean()" or "std()" and rename the data set to "fullData".
4. Convert the activity column to a factor, using labels describing the activities.
5. Change the labels of the data set 
6. Export data into tidy.txt
